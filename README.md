# bme590final_backend
Back end functionality for the HIE sensor management tool.
## Setting up Database
* create a databaseconfig.py file (this is in the gitignore and will not be added to git)
* in the file, add the following code:
  * mysql = {'host': [db host],
         'user': [user],
         'password': [password],
         'db': [name of db]}
